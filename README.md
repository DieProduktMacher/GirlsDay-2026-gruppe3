🎀 Girls Day – Tic Tac Toe Challenges
Welcome! Below are your three coding tasks. Read each one carefully, find the matching spot in the code, and make it your own. 🚀

Task 1 – Style the Board 🎨

Find the mystyle.css CSS class and change the following:

border-color → pick any color you like
border-width → make it thicker or thinner (e.g. 4px, 8px)
border-radius → round the corners if you want (e.g. 12px)

💡 Tip: Colors can be written as names (hotpink), hex codes (#ff69b4), or RGB (rgb(255, 105, 180)).


Task 2 – Customize the Symbols ✏️
Find where "X" and "O" are drawn and replace them with something fun.
Ideas:

Emojis → 🌸 vs ⭐, 🦄 vs 🐉, 🍕 vs 🍦
Letters → your initials
Words → "YES" vs "NO"

Task 3 – Check the winner logic 
Focus on the last 3 CSS selectors. They have a mini problem, since a hacker changed the logic and now it is very difficult to win the Tic Tac Toe game.
The hacker only told us that there are 10 problems that need to be fixed in order to win the game.
Viel Spaß!


💡 Don't forget to save the changes! :)
